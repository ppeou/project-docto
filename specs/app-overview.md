# App Overview - Project Docto

## Purpose
Project Docto is a Progressive Web App (PWA) designed for families to coordinate and manage healthcare for themselves or elderly parents. Family members can create shared healthcare itineraries, track appointments, manage prescriptions, and collaborate on care planning.

## Target Users
- **Primary Users**: Adult children managing care for elderly parents
- **Secondary Users**: Siblings, caregivers, family members coordinating healthcare
- **General Users**: Anyone managing their own or family member's healthcare

## Core Use Cases

### 1. Create Healthcare Itinerary
- A family member creates an itinerary (e.g., "Mom's Healthcare - Q1 2024")
- Sets itinerary name, date range, and description
- Defines patient information (name, relation)
- Becomes the creator/maintainer of the itinerary

### 2. Share Itinerary
- Creator shares itinerary with siblings/family members
- Invite via email or phone number
- Set access level (viewer or collaborator)
- Shared members can view and collaborate

### 3. Manage Doctor Appointments
- Add doctor appointments to itinerary
- Include: doctor name, specialty, contact info, clinic details
- Set date/time, duration, purpose
- Track appointment status (scheduled, completed, cancelled, rescheduled)
- Set reminders/notifications

### 4. Track Prescriptions
- Add prescriptions with medication details
- Track refill dates and reminders
- Link prescriptions to appointments or doctors
- Alert family members when refills are needed

### 5. Record Doctor Notes
- Add notes from doctor visits
- Attach to specific appointments
- Include test results, treatment plans, diagnoses
- Share important medical information with family

### 6. Family Collaboration
- Multiple family members can view shared itineraries
- View appointments and prescription schedules
- Receive notifications for upcoming events
- Coordinate care without constant communication
